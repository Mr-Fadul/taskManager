# TaskManager

Projeto Node.js inicializado com `npm init -y`.

## Como rodar

1. Instale as dependências (se houver):
   ```bash
   npm install
   ```
2. Execute o projeto:
   ```bash
   node index.js
   ```

## Sobre

Este é um projeto inicial para gerenciamento de tarefas usando Node.js.
